A Nice Breaking News or News Slideshow plugin for WordPress.

== Description ==

Nice Latest News Ticker is a Customizable responsive Jquery WordPress plugin that gives you a new experience with your website.

You will find a <a target="_blank" href="https://plugins-demo.nayeemriddhi.info/news-ticker/">Demo</a> here

<strong>Plugin Features </strong>

* Customizable responsive.
* All Browser support.
* Unlimited News Group Anywhere for use.
* Use via short-code.
* You Can Change Heading of News Ticker.
* You Can Sort With Up and Down Arrow.
* Add and remove News item from backend.
* News Link given option also available
* Highly recommended for User Experience.
* And Many More...
